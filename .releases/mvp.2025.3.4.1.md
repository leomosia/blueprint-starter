# mvp.2025.3.4.1

Introduces the first Blueprint TodoSys structure.

This release adds the execution visibility side of Blueprint: a lightweight flow for tracking actionable work without moving the underlying assets themselves.

Included:

- Added `_/systems/todosys/`
- Added six TodoSys stage folders
- Added a TodoSys README
- Added a story creation helper for TodoSys
- Updated the root README with the expanded systems direction

Not included yet:

- Detailed TodoSys templates
- Integration with project-level L5 folders
- Full handbook documentation
