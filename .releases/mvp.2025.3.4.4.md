# mvp.2025.3.4.4

Introduces public script examples and explainers.

This release makes Blueprint easier to understand by showing the helper scripts in action. The generated examples include provenance notes so users can see which command created each folder or file.

Included:

- Added public-safe InputSys stage README files.
- Added a starter `inbox.md` and whitelist-gated inbox splitter.
- Clarified that inbox whitelist tags represent approved project, body-of-work, and workstream destinations.
- Added routed inbox examples for `blueprint`, `bodyofwork`, and `project`.
- Added a script-generated content item example.
- Added a script-generated TodoSys story example.
- Added an L5 project structure example.
- Added a stats dashboard example.
- Added `BLUEPRINT_DATE` support for reproducible content, story, and stats examples.

Not included yet:

- Full year runtime example.
- OutputSys distribution example.
- Automated example reset command.
