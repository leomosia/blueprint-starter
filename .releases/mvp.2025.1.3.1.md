# mvp.2025.1.3.1

Introduces the first Blueprint InputSys and OutputSys structures.

This release adds the content and knowledge system layer of Blueprint. InputSys handles the path from raw material to usable building blocks. OutputSys handles the path from finished assets to distribution, engagement, feedback, and long-term reuse.

Included:

- Added `_/systems/inputsys/`
- Added the eight InputSys stage folders
- Added an InputSys README
- Added `_/systems/outputsys/`
- Added the eight OutputSys stage folders
- Added an OutputSys README
- Added script PATH setup support for local Blueprint helper commands
- Added helper scripts for content item creation, seeding, movement, OutputSys handoff, and local stats
- Updated the root README with the systems direction

Not included yet:

- Detailed InputSys templates
- Detailed OutputSys templates
- Detailed automation inside each stage
- Full handbook documentation
