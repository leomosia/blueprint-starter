# mvp.2025.3.4.2

Introduces the first Blueprint BacklogSys structure.

This release adds a lightweight manual backlog for possible work that is worth keeping but not ready to enter TodoSys yet.

Included:

- Added `_/systems/backlogsys/`
- Added `backlog.md` for manual backlog headings
- Added a BacklogSys README
- Updated the root README with the expanded systems direction

Not included yet:

- Backlog automation
- Backlog-to-TodoSys promotion helpers
- Detailed backlog templates
